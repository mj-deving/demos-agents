What you can say safely

  • The repo ships a real consumer package with maintained validation paths.
  • The archetype checks are current and reproducible from repo source.
  • The research-agent publish path is proven on the production host, with delayed indexer confirmation.
  • Reply/react, market writes, and official identity linking are also live-proven on the current host.
  • Attestation source-chain quality has a maintained stress path before live publish use.
  • The docs, playbooks, and generated skill artifacts all live in one canonical repo.

What you should not oversell yet

  • That npm install is already the default public path.
  • That every publish will become indexed-visible within the shorter default probe window.
  • That the market-analyst starter has a second full live publish proof without qualification.
  • That all live write families have equally strong readback evidence.
  • That getPriceHistory() is populated enough on the production host to drive launch-critical strategy decisions.
